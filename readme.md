  <code>npm install</code><br><br>

</section>

  <code>PORT=5000</code>
</section>

<section>
  <h3>Installation Step 03</h3>
  <code>npm run dev</code><br><br>
  <p>Afte installation of all dependencies & configuration run code using npm</p>
</section>
